이틀전까지 한 내용은 태그 : div, p, h1~h6, span, br, img, table, a 스타일을 줘보자 - 속성(attribute) style을 주자
style 속성 = CSS(cascading style sheets)
width : 너비, max-width, min-width
height : 높이, max-height, min-height
background-color : 배경색, 색상을 태그에 입힐 수 있다, rgb(255,255,255);
padding(태그 내부 여백), margin(태그 외부 여백) : top, right, bottom, left
값을 한개만 썼을때는 사방면이 그 값으로 적용된다.
각각 지정하고 싶을땐 4개 값 모두 작성 해줘도 된다.
값을 두개 작성했을때, 상하 좌우 값을 적용할 수 있다.
box-sizing : 박스의 크기를 어떻게 설정할지 기준을 설정
속성을 추가 할때 속성 명 = "속성 값" display : 영역의 노출 크기를 어떻게 설정할지 기준을 설정한다. 부모 태그를 기준으로 모든 너비를 차지한다. 높이는 자식의 기준으로 한다.
