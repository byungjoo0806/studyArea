# 포켓몬 뽑기 컨트랙트

- 리액트 설치
```sh
npx create-react-app erc20
```

- 컨트랙트 작성 후
```sh
npm install -g @remix-project/remixd
remixd -s . --remix-ide https://remix.ethereum.org
```

- 가나쉬 테스트 네트워크 열기
```sh
npm i -g ganache-cli
npx ganache-cli --chain.chainId 1337 --chain.networkId 1337
```
